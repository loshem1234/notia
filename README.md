# Notia

A frictionless, self-organizing idea capture system.

## Phase 1 MVP Features

- 🎤 Instant voice/text capture via home screen widget
- 💾 Local-first storage (SQLite)
- 🤖 AI categorization (Claude via Railway backend)
- 📋 Browse & search notes
- ⚡ Zero friction: tap, talk, done

## Tech Stack

- **Frontend**: Flutter (iOS/Android)
- **Backend**: Python FastAPI on Railway
- **Database**: SQLite (local) + PostgreSQL (cloud backup)
- **AI**: Claude API (Anthropic)

## Project Structure
